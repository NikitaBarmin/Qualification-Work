Важный новый этап
Функционал ИИ-помощника:
•	Плавающий чат в правом нижнем углу на странице дашборда
•	Имеет полный контекст текущего анализа (метрики, графики, SWOT, описание бизнеса)
•	Может отвечать на вопросы вида:
o	«Почему упала маржинальность?»
o	«Что мне делать дальше?»
o	«Сравни этот месяц с прошлым»
o	«Составь план действий на квартал»
Техническая реализация:
•	Отдельный сервисна бэкенде
•	Передача в AI API (OpenAI / Claude / Grok) промпта, который включает:
o	Описание бизнеса (из формы)
o	Ключевые метрики
o	Данные SWOT
o	Текущий запрос пользователя
•	Ответ сохраняется и отображается в чате
Ограничения:
•	ИИ не должен «выдумывать» данные, которых нет в анализе
•	Должен работать только в рамках загруженного датасета
