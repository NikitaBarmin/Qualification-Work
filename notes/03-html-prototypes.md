ЭТАП 3: HTML-прототипы (Low → High Fidelity)
•	Low-fidelity (wireframes) — для утверждения структуры
•	Mid-fidelity — для утверждения компоновки
•	High-fidelity — финальные дизайны всех ключевых экранов
Необходимые экраны для прототипирования:
•	Главная страница
•	«Как это работает» (демо)
•	Мои датасеты
•	Новая аналитика (пустое + с загруженным файлом)
•	Аналитика (полноценный дашборд)
•	Страница авторизации / регистрации
•	Настройки
•	Пустые состояния
