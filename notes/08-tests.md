1. Инструменты
Vitest — runner для юнит- и интеграционных тестов (работает с Vite).
React Testing Library (RTL) + userEvent — для тестирования React-компонентов (эмуляция действий пользователя).
Playwright — для E2E-тестов (реальный браузер, полные сценарии).
vitest --coverage (v8) — проверка покрытия кода.
